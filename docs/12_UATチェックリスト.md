# 資産管理システム UAT観点チェックリスト

## 1. 認証
- [ ] ログイン成功でダッシュボードへ遷移する
- [ ] 連続失敗5回でロックされる
- [ ] 未ログイン時に業務画面へアクセスできない

## 2. 資産管理
- [ ] 資産登録が成功する（予算未紐付け許容）
- [ ] PC資産で利用者未設定時に登録失敗する
- [ ] 予算後付け紐付けが成功する

## 3. 構成/予算/ライセンス
- [ ] 構成登録が成功する
- [ ] 廃棄済資産の構成追加が拒否される
- [ ] 予算登録/執行済予算登録が成功する
- [ ] ライセンス割当上限超過が拒否される

## 4. PC/棚卸
- [ ] PC割当登録が成功する（PC資産のみ）
- [ ] 棚卸結果登録/更新が成功する

## 5. CSV・バッチ
- [ ] 資産CSVエクスポートが取得できる
- [ ] 資産CSVインポートで正常行が取り込まれる
- [ ] 日次バッチ3種が正常終了する
- [ ] 棚卸スナップショットバッチが正常終了する

## 6. 監査
- [ ] 主要更新操作が監査ログに記録される
- [ ] 監査ログ画面で追跡可能である

## 7. 機能-画面対応の受入確認
- [ ] F-00（認証/ユーザ管理）が SCR-000（ログイン）で成立する
- [ ] F-01（ダッシュボード可視化）が SCR-001（ダッシュボード）で成立する
- [ ] F-02（資産管理）が SCR-010/SCR-011 で成立する
- [ ] F-03（構成管理）が SCR-020 で成立する
- [ ] F-04（予算/執行済予算管理）が SCR-030 で成立する
- [ ] F-05（ライセンス管理）が SCR-040 で成立する
- [ ] F-06（PC管理）が SCR-050 で成立する
- [ ] F-07（棚卸管理）が SCR-060 で成立する
- [ ] F-08（廃棄承認）が SCR-070 で成立する
- [ ] F-09（監査ログ閲覧）が SCR-080 で成立する
